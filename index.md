# AXSDK Sites

> Site-specific `llms.txt` sitemap data implemented for the AXSDK Chrome extension.

## Sites

- [amazon](https://www.amazon.com): 상품 검색, 상세 조회, 옵션/수량 선택, 장바구니 추가·조회·수량 변경·삭제, 결제(checkout) 진행을 할 수 있습니다.
- [bluemoonsoft](http://bluemoonsoft.com): 회사·제품·보안 모듈 정보를 탐색하고 견적 문의 폼을 조회·입력·제출할 수 있습니다.
- [thumbtack](https://www.thumbtack.com): 주소에서 ZIP을 확인해 지역 서비스 전문가를 검색하고, 프로필·리뷰·서비스 정보를 조회하며 견적 요청 폼을 열고 입력할 수 있습니다.
