# DOCURAY 제품 라인업

블루문소프트의 DOCURAY는 4개의 제품 라인업으로 구성되며, 사용 환경과 요구사항에 따라 선택할 수 있습니다.

## 1. Docuray Office DRM

| 항목 | 내용 |
|---|---|
| 대상 파일 | MS Word, Excel, PowerPoint, 아래아한글, PDF |
| 핵심 기능 | 자동 암복호화, 로그인만으로 기존 환경과 동일한 사용 경험 |
| 특징 | 도입 후 별도 학습 없이 사용 가능 |

## 2. Docuray CAD DRM

| 항목 | 내용 |
|---|---|
| 대상 | CAD 프로그램이 사용하는 프로젝트, 도면, 조립품, 부품 등 |
| 지원 CAD | AutoCAD, SolidWorks, CATIA, Inventor 등 |
| 핵심 기능 | 자동 암복호화 |
| 특징 | 기존 환경 그대로 사용 — 도입 용이, 사용 편리 |

## 3. Docuray Privacy DRM

| 항목 | 내용 |
|---|---|
| 목적 | 개인정보보호법 대응 |
| 동작 방식 | 설정된 키워드/패턴에 따라 문서 내용을 검토 |
| 처리 액션 | 완전삭제 / 격리 / 문서 내 텍스트만 추출 기록 |

## 4. Docuray with CodeGuard

| 항목 | 내용 |
|---|---|
| 대상 | PC 및 서버 내 모든 소스코드 |
| 지원 IDE | Visual Studio, Eclipse, Android Studio 등 |
| 지원 언어 | C, C++, C#, Java, Python 등 |
| 형상관리 | GIT, SVN 등 |
| 특징 | PC뿐 아니라 서버 소스코드까지 암호화 가능 |
