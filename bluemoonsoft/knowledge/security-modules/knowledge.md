# DOCURAY 보안 모듈 (16종)

|---|---|---|---|---|
| 01 | 랜섬 제로 | Ransom Zero | 랜섬웨어 | 랜섬웨어 차단부터 다중협박 대응 및 파일 복구까지 한 번에 해결 가능한 솔루션 |

| 03 | 도면 보안 | CAD Guard | 파일 암호화 | AutoCAD, SolidWorks, CATIA, Inventor 등 다양한 도면 암호화 보호 |
| 04 | 이미지 보안 | Image Guard | 파일 암호화 | Adobe Photoshop, Illustrator 등 다양한 이미지 파일 암호화 보호 |
| 05 | 소스코드 보안 | Code Guard | 파일 암호화 | Visual Studio, Eclipse, Android Studio 등 IDE와 C/C++/C#/Java/Python 등 다양한 언어 지원, GIT·SVN 등 형상관리 도구 지원 |
| 06 | 프로세스 보안 | Process Guard | 행위 통제 | 위협 프로세스 차단, 사내 실행 프로세스 기록·통계 제공 |

| 09 | 출력물 보안 | Print Guard | 유출 식별 | 출력 워터마크와 출력 결과 확인을 통한 유출자 식별 |
| 10 | 개인정보 보안 | Privacy Guard | 컴플라이언스 | 문서 내 개인정보 탐지 및 위험치 자동 측정을 통한 강제 삭제 |
| 11 | USB 보안 | USB Guard | 매체 통제 | 이동식 매체 연결 통제로 유출 차단, 사용 가능 매체에 대해 파일 생성 감사 로그 |
| 12 | 메신저 보안 | Messenger Guard | 채널 통제 | 카카오톡, 스카이프, 위챗, 텔레그램, 네이트온, 라인, 밴드 등 메신저 파일 유출 차단·기록 |

| 15 | 모바일 뷰어 | Mobile Viewer | 모바일 | 인증 사용자가 핸드폰·태블릿 등으로 암호화 문서 조회 |
| 16 | 엑스가드 | eXe Guard | 실행파일 보안 | 실행파일 암호화로 리버스엔지니어링·해킹 방어 |
